How to Install

  1. Download the .apk file by clicking the button above.
  2. Open your device's "Files" or "My Files" app and find the downloaded file.
  3. Tap the file to begin installation.
  4. You may need to grant permission to install from "Unknown Sources." If prompted, tap "Settings" and enable the permission for your browser or file manager.
  5. Return to the installation and complete it.
